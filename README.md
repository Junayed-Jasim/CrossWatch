# CrossWatch - Activity Logger

## What is this?
A simple Python program that tracks which applications you use on your computer.

## Features
-  Tracks active application every 5 seconds
-  Saves data to CSV format
-  Shows window titles
-  Handles special characters safely

## How to Run
1. Install Python
2. Install requirements: `pip install psutil pywin32`
3. Run: `python logger.py`
4. Press CTRL+C to stop
